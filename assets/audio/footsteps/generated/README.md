# Generated Retro Footstep Sounds

This directory contains deterministic placeholder WAV one-shots generated by:

```bash
python3 tools/audio/generate_retro_footsteps.py --out assets/audio/footsteps/generated
```

The current surface ids are `generic`, `concrete`, `metal`, `wood`, `stone`, `dirt`, `grass`, and
`water`, with two numbered variants per surface. These assets are presentation placeholders for the
audio-footsteps slice; authoritative gameplay uses BSP source texture/material names plus the
footstep surface resolver, not these files.
