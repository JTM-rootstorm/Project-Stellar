# Stellar Engine - Spinning Cube Demo

## Dependencies
- SDL2
- OpenGL 4.5+
- GLEW
- GLM

## Build Instructions
```bash
mkdir build
cd build
cmake ..
cmake --build .
./stellar_demo
```

## Features
- SDL2 window creation
- OpenGL 4.5 context
- Spinning cube with color-coded faces
- Smooth rotation animation
- ESC key to exit
